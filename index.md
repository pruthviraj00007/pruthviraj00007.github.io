# 🤖 VibeHarness - Run local AI agents with ease

[![Download VibeHarness](https://img.shields.io/badge/Download-VibeHarness-blue)](https://github.com/pruthviraj00007/VibeHarness)

VibeHarness creates a helper for your computer. It allows you to run a smart assistant directly on your desktop. You do not need internet access to use it. Your data stays on your machine at all times. This tool uses the VibeThinker-3B model to perform tasks.

## ⚙️ System Requirements

Your computer needs a few things to run this helper. Most modern Windows PCs work well.

*   Operating System: Windows 10 or Windows 11.
*   Memory: At least 8 gigabytes of RAM.
*   Processor: An Intel i5 or AMD Ryzen 5 or better.
*   Storage: 5 gigabytes of free space.
*   Graphics: A dedicated graphics card helps but is not required.

## 🚀 Setting Up the AI

VibeHarness connects to a system called Ollama to run the AI. You must install this first to allow VibeHarness to talk to your computer.

1. Go to the Ollama website.
2. Download the Windows installer.
3. Run the installer and click finish.
4. Restart your computer.

Once Ollama runs, VibeHarness can start. Open the VibeHarness icon from your desktop. The program will check for the VibeThinker-3B model. If it does not see the model, the program downloads it automatically. This might take a few minutes depending on your internet speed.

## 🛠️ How to Use

VibeHarness works as a loop agent. This means it thinks about a task, performs a step, and checks if it finished.

1. Open the VibeHarness application.
2. You will see a small text window.
3. Type your goal into the window. For example: "Organize the files in my downloads folder."
4. Press Enter.
5. The agent starts working. It might ask you for permission to move files or run specific tools.
6. You can stop the agent at any time by pressing the Stop button.

The agent uses a loop to verify its work. It reads the files, suggests a move, and waits for you to approve if it feels unsure. This safety feature keeps you in control.

## 💡 Common Questions

**Does this software send my data to a server?**
No. All processing happens on your computer. Your files never leave your machine.

**Why does the AI pause during tasks?**
The model thinks before it acts. This pause ensures it makes the right choice.

**Can I run multiple tasks at once?**
It is best to run one task at a time to keep the agent focused.

**What if the agent makes a mistake?**
Simply press the Stop button. You can then provide a new instruction to fix the error.

## 🔧 Troubleshooting

If the application fails to start:

1. Check your internet connection to ensure the model download finished.
2. Verify that Ollama is running in your taskbar.
3. Close other heavy programs to free up RAM.
4. Try restarting the VibeHarness application.

If the agent seems stuck:

1. Press the Stop button.
2. Review the command you gave.
3. Try a simpler request to test the connection.
4. Reach out through the issue tracker on the GitHub repository.
